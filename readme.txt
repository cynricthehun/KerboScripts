## How to transfer Kerboscript to your kerbal installation.
1. Store scripts in /KerbalInstallationFolder/Ships/Script
2. Launch the game.
3. Open your console on the ship you want to execute the script on.
4. Copy the script file from archive (0) by running ``` COPYPATH("0:/KerboScript.ks", ""). ```
5. You can check to see if the file is in your local volume now by running ``` list files. ```
6. If you successfully have the script in your ship you can execute/run it by using ``` run KerboScript ```

## Standard Commands
- SWITCH TO
- COPYPATH
- RUN
- LIST
