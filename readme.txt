## How to transfer Kerboscript to your kerbal installation.
- Store scripts in /KerbalInstallationFolder/Ships/Script
- Launch the game.
- Open your console on the ship you want to execute the script on.
- Copy the script file from archive (0) by running ``` COPYPATH("0:/KerboScript.ks", ""). ```
- You can check to see if the file is in your local volume now by running ``` list files. ```
- If you successfully have the script in your ship you can execute/run it by using ``` run KerboScript ```

## Standard Commands
- SWITCH TO
- COPYPATH
- RUN
- LIST
